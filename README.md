# Template Slim Framework - Versão Completa

## 📋 Sobre
Este é um template COMPLETO do Slim Framework, incluindo todas as dependências na pasta `vendor/`.
Ideal para laboratórios que NÃO possuem Composer instalado.

## 🚀 Como usar (NÃO PRECISA DE COMPOSER!)

1. **Baixe** este template completo
2. **Copie** para seu ambiente de laboratório
3. **Inicie** o servidor PHP embutido:

```bash
c:\xampp\php\php.exe -S localhost:8080 -t public/
