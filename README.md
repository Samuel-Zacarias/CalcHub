# CalcHub

CalcHub é uma coleção de calculadoras simples desenvolvida como projeto de estudo, reunindo diferentes ferramentas de cálculo em uma única interface. O objetivo é praticar lógica de programação, manipulação de formulários e desenvolvimento front-end.

## ✨ Funcionalidades

| Calculadora | Descrição |
|---|---|
| 📈 Nota | Calcula a média e apresenta a situação do aluno |
| 💪 IMC | Calcula o IMC e apresenta a classificação |
| 👩‍🚒 Funcionário | Calcula o salário com horas extras |
| 🎁 Produto | Cadastra produtos com preço, quantidade e saída |
| 📐 Triângulo | Calcula a área e classifica o tipo de triângulo |
